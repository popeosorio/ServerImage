Bienvenido.

En este repositorio encontraras un proyecto de carga de imagenes para usuarios registrados y visualizacion y busqueda de las mismas para usuarios no registrados:

Instalación.

Para instalar solo es necesario descargar el proyecto y colocarlo en el directorio publico del servidor que este usando, debe modificar la siguiente linea de codigo si desea cambiar la carpeta donde se almacenan las imagenes dentro del archivo model.php

$destination_dir="uploads";

...

$directorio = "uploads/".$img_file;
